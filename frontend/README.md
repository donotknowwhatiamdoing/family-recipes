# Frontend (React + Vite)

## Start lokal

1. `.env.example` zu `.env` kopieren.
2. Optional `VITE_API_BASE_URL` setzen (z.B. `http://localhost:8000`).
3. Abhaengigkeiten installieren und starten:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Der Build liegt danach in `dist/` und kann auf all-inkl als statische Seite deployed werden.
