# Familien-Rezeptesammlung

Dieses Repository enthält unsere gemeinsame Sammlung von Familienrezepten.

## Ziel

- Rezepte an einem Ort sammeln
- Änderungen nachvollziehbar versionieren
- Einfach gemeinsam bearbeiten

## Struktur (Vorschlag)

```text
recipes/
  vorspeisen/
  hauptgerichte/
  desserts/
assets/
  images/
```

## Einstieg

1. Repository klonen:
   ```bash
   git clone https://github.com/donotknowwhatiamdoing/family-recipes.git
   ```
2. In den Ordner wechseln:
   ```bash
   cd family-recipes
   ```
3. Neue Rezepte hinzufügen und committen:
   ```bash
   git add .
   git commit -m "Rezept hinzugefügt: <name>"
   git push
   ```

## Beitragen

- Möglichst ein Rezept pro Commit
- Aussagekräftige Commit-Nachrichten verwenden
- Bei Änderungen bestehender Rezepte kurz im Commit-Text beschreiben, was angepasst wurde
